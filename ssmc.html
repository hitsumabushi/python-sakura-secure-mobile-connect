class ssmc.PagingInfo(total=None, size=None, offset=None)[ソース]¶

ベースクラス: object

ページングされた情報をハンドルするための共通クラスです。 あるAPIについて、ページングされた内容をすべて取得したい場合には以下のようにして、PagingInfoをチェックすることで可能です。:

while True:
    _, paging = api.some_request(size=size, offset=offset)
    total = paging.total
    offset = paging.size + paging.offset
    if offset >= total:
        break
